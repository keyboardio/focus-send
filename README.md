## About

A small library to interface with [Kaleidoscope][kaleidoscope] powered keyboards via [Focus][focus], including a few example tools, among them a port of the [focus-send][focus-send] tool to Rust.

 [focus-send]: https://github.com/keyboardio/Kaleidoscope/blob/master/bin/focus-send
 [kaleidoscope]: https://github.com/keyboardio/Kaleidoscope

See the `focus-send --help` output for documentation.
